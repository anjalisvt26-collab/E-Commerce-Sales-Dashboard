 E-Commerce Sales Dashboard (Power BI)

Project Overview

This project is an interactive Power BI dashboard developed to analyze and visualize e-commerce sales data. It provides valuable insights into sales performance, profitability, customer behavior, product categories, payment methods, and regional sales trends.

 Dataset

**Orders.csv**
- Order ID
- Order Date
- Customer Name
- State
- Category
- Sub-Category
- Sales Amount
- Quantity

**Details.csv**
- Order ID
- Profit
- Payment Mode

 Dashboard Features

- KPI Cards for Total Sales, Profit, and Quantity
- State-wise Sales Analysis
- Monthly Profit Analysis
- Customer-wise Sales Analysis
- Category-wise Quantity Distribution
- Payment Mode Analysis
- Sub-Category Profit Analysis
- Quarter-wise Interactive Filter

Key Performance Indicators

- **Total Sales:** ₹438K
- **Total Profit:** ₹37K
- **Total Quantity Sold:** 6K

 Key Insights

- Maharashtra generated the highest sales.
- December recorded the highest profit.
- Clothing was the highest-selling category.
- Cash on Delivery (COD) was the most preferred payment method.
- Printers generated the highest profit among all sub-categories.

  Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
